# vanilla-rnn-machine-translation

This project presents an English-to-Urdu neural machine translation system built with a vanilla RNN encoder-decoder architecture.

The notebook covers:
- dataset loading and preprocessing
- train, validation, and test splitting
- tokenization and vocabulary construction
- sequence encoding, padding, and batching
- vanilla RNN model implementation and training
- greedy decoding and beam search
- BLEU-based evaluation and error analysis

The implementation is developed in PyTorch and focuses on demonstrating the full machine translation pipeline using a basic recurrent architecture.
